**You will need a PowerBI business account to interact with this dashboard**

This dashboard was created as a test for a small business. The company manufactures and sells a range of items. The purpose of this PowerBI project was to track inventory, sales, utilization, and schedules. Below is a collection of images and descriptions for each page of this project.

## Main Inventory
Here I have visualized inventory data over a four month span. The top row shows total inventory, value of all items, items manufactured, items sold, and total profit. There is a slicer which allows filtering of specific dates, items, and item codes. The three graphs are used to visualize inventory stock and price per item.

 ## Process Utilization
 This page is used to show the production utilization of each machine in operation. Here we can see how much time each machine takes to create one item per piece and step. The data and visualizations can be filtered by machine name or ID. This is useful to see how to effectively schedule a workday or workweek depending on the amount of time for each machine to produce an item. It can also help to prioritize if needed one machine over another.

## Sales Stats
This page shows all sales information including transactions, total items sold, item types sold, and profit. Slicer can be used to filter by date, item, and customer. Graphs are used to visualize sales by item, customer total purchases, and daily profit. An additional slicer has also been added to re-calculate profit based on price adjustment. This can be useful if stakeholders and investors would like to see how raising or lowering the price of each item would affect profit.

## Production Schedule
This final page shows the importance of maintaining a balance of supply and demand. The production schedule main purpose is to present data on raw material inventory to determine if it meets the demand for production. The orders and raw materials are linked using a network of excel spreadsheets which are auto updated in the data presented on the dashboard page. As orders are placed, raw materials are removed from the inventory. If the orders exceed what can be produced then the table shows there is not enough raw material in stock. This can change based on the order date. This page also shows daily production output of each item, production efficiency (if we met our order date), and general price per item.
